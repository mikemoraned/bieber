What is a Bieber?

Some people have recently been using a 'bieber' as a unit of time, based on Mr Biebers current age (). I call this the Mr Bieber definition.

To illustrate, if you are years old, then by the above definition, you are around biebers old.

I far prefer a more experimentally determined definition. Currently, the word "bieber" is mentioned on twitter roughly once every seconds. Let's say this corresponds to a single bieber-unit. We can then build up larger time intervals using standard SI units. I call this the SI Bieber definition.
